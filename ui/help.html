Terminal commands

    clear                  Clear terminal window with linebreaks.
    exit                   Exit the terminal (pop-up window only).
    fetch                  Fetch responses from interpreter.
    help, h, ?             Show this menu.
    info                   Display session info.
    reset                  Reset terminal window.
    startautofetching      Set automatic fetch rate of this terminal.
      <MIN SECONDS> <MAX SECONDS>
    stopautofetching       Stop automatically fetching new packets.
    xss <CODE>             Execute JavaScript in this terminal.
